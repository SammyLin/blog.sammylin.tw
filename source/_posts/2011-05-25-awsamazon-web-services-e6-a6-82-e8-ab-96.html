什麼是雲端？在這邊就不多說了，網路上怎麼Google都可以找到很多資料，這邊要介紹的是怎麼樣才能坐享雲端的世界。雖然Amazon在上個星期發生了大斷線，不過對廣大的對新技術的好奇者來說不影響對他研究的熱忱，包括我也是~呵~~。

AWS的功能在目前來講(2011/5/25)已經有好幾項功能了，往後可能還會新增也說不定，至於新的專有名稱也很多(是要創造Amazon語言嗎？@@)

運算服務

  • Amazon Elastic Compute Cloud(EC2)-Amazon 彈性雲端
  • Amazon Elastic MapReduce-Amazon 雲端數據分析
  • Auto Scaling-自動延展
網路服務
  • Amazon Route 53-Amazon 53號路由器
  • Amazon Virtual Private Cloud(VPC)-Amazon 虛擬私有雲
  • Elastic Load Balancing-彈性負載平衡
  • AWS Direct Connect-AWS 專屬連結
內容交付服務
  • Amazon CloudFront-Amazon 內容傳遞網路(CDN)
部署與管理
  • AWS Elastic Beanstalk-AWS 自動管理雲端佈署
  • AWS CloudFormation-Armazon 雲端小精靈
儲存服務
  • Amazon Simple Storage Service(S3)-Amazon 簡易儲存系統
  • Amazon Elastic Block Store(ESB)-Amazon 彈性儲存空間
  • AWS Import/Export-AWS輸入/輸出
資料庫服務
  • Amazon SimpleDB-Amazon 簡易資料庫
  • Amazon Relational Database Service(RDS)-Amazon 關連性資料庫
  • Amazon ElastiCache-Amazon 彈性快取
監測服務
  • Amazon CloudWatch-Amazon 雲端資源監控管理
支付服務
  • Amazon Flexible Payments Service(FPS)-Amazon 彈性付費服務
  • Amazon DevPay-Amazon 開發者付費服務
訊息服務
  • Amazon Simple Queue Service(SQS)-Amazon 簡易佇列服務
  • Amazon Simple Notification Service(SNS)-Amazon 簡易通知服務
  • Amazon Simple Email Service (SES)-Amazon 簡易郵件服務
電子商務
  • Amazon Fulfillment Web Service (FWS)-Amazon 商務實現服務
其他
  • AWS Identity and Access Management (IAM)-AWS 身份識別與存取管理
