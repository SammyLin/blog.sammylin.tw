今天用Jquery抓Textbox的值時，發現有個問題

抓出來變成亂碼這樣：

這才發生原來我的jquey出了問題

$("#AddTagJsonBtn").click(function () {
        alert($('#TagText').val);
    });

 

看來就是val沒有加括號而以，居然讀出來變這樣。

正確應該是要像下面那樣：

$("#AddTagJsonBtn").click(function () {
        alert($('#TagText').val
());
});
