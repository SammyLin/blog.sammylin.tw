[Rails]安裝devise在mongo_mapper

Dec 3rd, 2011 | Comments

參考了XDite的Rails 101慢慢的上手了，而我希望可以配合MongoDB來建立我的網站不過還有些地方跟Rails 101不大一樣，所以來筆記一下了

使用者機制在Rails很快就可以完成，因為我們可以直接用別人開發過的Plugin，就可以懶懶的完成你所要的功能了，在Rails 101上面推薦的使用者Plugin是Devise

修改Gemfile

安裝新的plugin

$ bundle install

產生Devise檔案

$ rails generate devise:install

修改config/initializers/devise.rb 配置

建立users的model

rails generate devise users

其他的就沒什麼差別了~
