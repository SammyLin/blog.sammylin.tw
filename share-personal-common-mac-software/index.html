[分享]個人常用OSX(MAC)軟體

| Comments

其實我買Macbook才不到兩三個月的時間，為了找適合自己的軟體其實找了很累人就是了，有些是免費的，不過還有一些軟體是付費的，這兩者其實還是有差別的，有的軟體還是需要付費的還是比較好用(天下沒有白吃的午餐咩)，而且為了往後我電腦不小心掛點，我決定寫一篇文章來提醒自己這些是我找的還不錯的軟體。

每個人有每個適合用的軟體，如果有推薦或是更好用的軟體歡迎分享，畢竟我是個Apple的新手~~哈哈

1.離線編輯

Ecto

軟體名稱：Ecto 費用：免費
官方網站：http://illuminex.com/ecto/ 簡單說明 ：因為過去我在PC上面都是用Windows Live Writer來編輯的，到了Mac，離線編輯的軟體很多，我選了ecto，用離線編輯最大的主因其實就是我常常會把螢幕截取下來貼到文章裡面，所以在線上編輯我就會變的很麻煩，還要一直上傳照片，所以我才愛用離線編輯器來寫我的部落格。

2.螢幕截取

軟體名稱：LittleSnapper
費用：免費 官方網站：http://www.realmacsoftware.com/littlesnapper/ 簡單說明 ：Mac其實有內建的螢幕截取，但是我的圖片裡面可能會做一些文字編輯或是把我的重要隱私馬賽克，LittleSnapper可以方便的做到這一點，而且可以幫你做整理，分門別類的把過去拍下來的圖片做一個整理。 但是免費版的只能拍30張照片，不過用完之後把他刪掉就可以繼續拍了~

3.批次處理圖片

軟體名稱：QuickScale
費用：$5.99(美金) 官方網站：http://www.codingmammoth.com/quickscale.php 簡單說明 ：因為我遇過之前我的文章被對岸的朋友拿去使用又沒有告之，而且還大剌剌的在我的圖片上貼了他們網站的浮水印，所以之後我的圖片大部份都會做浮水印。QuickScale 可以批次幫你的圖片做縮放及上浮水印。

4.錄製螢幕

軟體名稱：ScreenFlow
費用：$99.99(美金) 官方網站：http://www.telestream.net/screen-flow/ 簡單說明 ：這個軟體是貴了一點，不過貴的很值得，一般的軟體錄製之後就直接做匯出，但是ScreenFlow可以做後製，可以幫你的影片做浮水印或是幫你重要的隱私做模糊化。很酷的是他可以一邊做螢幕的錄製還可以視訊錄製，如果不是很害羞的人可以把你的大頭放在你的右下角(我是很害羞啦~xd)

5.遠端連線

軟體名稱：Remote Desktop Connection
費用：免費 官方網站：http://www.microsoft.com/downloads/zh-tw/details.aspx?familyid=68346e0d-44d3-4065-99bb-b664b27ee1f0&displaylang=zh-tw 簡單說明 ：要連線到PC的遠端桌面這個是獨一無二的選擇了。

軟體名稱：TeamViewer
費用：免費 官方網站：http://www.teamviewer.com 簡單說明 ：很簡單就可以遠端連線到別人的桌面，也有出Windows的版本，所以不用怕拿到mac就連不到別人的桌面了。

6.資料庫管理GUI

軟體名稱：Navicat Lite
費用： 免費
官方網站：http://navicat.de/cht 簡單說明 ：因為資料庫不是我的專業，如果有軟體可以輔助我應該就是Navicat你可以連線到Mysql ,SQL Server ,SQLite ,Oracle ,PostgreSQL。目前Navicat似乎中止了Navicat Lite的服務，如果你要下載的話，請到這裡http://www.macupdate.com/app/mac/32916/navicat-lite，

7.心智圖

軟體名稱：XMind
費用：免費 官方網站：http://www.xmind.net/ 簡單說明 ：一般來說畫心智圖，XMind已經很完整了，而且他也有Windows的版本，可以互相共用。

8.筆記軟體

軟體名稱：Evernote
費用：免費 官方網站：http://www.evernote.com/ 簡單說明 ：用了很久的筆記本軟體，我都用到中途就不想用或是忘記他，Evernote你可以把你的筆記上Tag，分類，也可以用搜尋把你過去的筆記找出來，更可以分享你的筆記給別人。

軟體名稱：Wunderlist
費用：免費 官方網站：http://www.6wunderkinder.com/wunderlist/ 簡單說明 ：這是個好用的待辦資料，簡單明瞭，沒什麼多餘的功能，所以才讓我愛上XD。

9.空間

軟體名稱：Dropbox
費用：免費 官方網站：http://www.dropbox.com/ 簡單說明 ：也是個簡單好用的雲端空間，Windows或是Mac都可以使用，免費版的有2G的空間

10.即時通

軟體名稱：Skype
費用：免費 官方網站：http://www.skype.com/ 簡單說明 ：從網路電話起家，可以用訊息傳遞也能打電話，不過好像有很多家都可以打網路電話，不過還是Skype佔大宗。

軟體名稱：adium
費用：免費 官方網站：http://adium.im/ 簡單說明 ：可以整合你的即時通軟體，msn、yahoo、gtalk…等等的。

11.終端機

軟體名稱：iTerm 官方網站：http://iterm.sourceforge.net/ 簡單說明 ：還蠻有名的終端機程式

12.簡報

軟體名稱：Keynote
費用：$19.99(美金) 官方網站：http://www.apple.com/tw/iwork/keynote/ 簡單說明 ：Keynote是apple旗下的產品，可能是apple自己的對於編輯slide還蠻順手的，而且可以轉成PPT檔。

軟體名稱：PowerPoint
費用：$4190(新台幣) 官方網站：http://www.microsoft.com/taiwan/mac 簡單說明 ：微軟的簡報軟體，如果是常用PowerPoint也是很容易上手的。

13.休閒軟體

軟體名稱：PPS
費用：免費 官方網站：http://www.pps.tv/ 簡單說明 ：休息的時候可以看的一個網路電視。
