Amazon S3 己經擁有449億個物件

| Comments

在2011年第二季結算下來，Amazon S3已經擁有了449億個物件以及在尖峰時間中每秒29萬個requests。

不管企業怎麼樣想，雲端是多麼不安全，不穩定，之前我還聽到一個笑話，還是一個總經理嘴巴講出來的他說：「雲端發展不完全，也沒有很流行。」其實這是我一個好友去面試時從那個總經理嘴巴講出來的，不然的話應該會想當面吐他槽的。

其實不是雲端不安全也不穩定，其實雲端發展已經很建全了，只是看你怎麼使用而以，而且S3可以用API把你的檔案在本地加密之後上傳了，在S3上面的檔案也就是加過密的資料，金錀你就可以留在本地端，或者是說，把你的檔案分機密、普通，可以把你普通一般重要的資料就可以放到雲端上面去。

[ASP.NET MVC]輸出HTML

| Comments

原來在ASP.NET MVC 要輸出文字時，會幫你自動的轉成可以在HTML顯示的符號，很聰明，但是剛剛遇到一個問題就是我是要輸出一個JS，所以裡面有單引號跟雙引號，就是不想讓他輸出成HTML顯示的文字，一般來說我們要將文字從Controller輸出至View時應該會這樣的：

Controller

   1: public ActionResult test()

   2: {

   3:    StringBuilder str = new StringBuilder();

   4:     str.Append("<script type=\"text/javascript\">");

   5:     str.Append("alert('我要顯示出來~~~');");

   6:     str.Append("</script>");

   7:     ViewBag.Str = str.ToString();

   8:     return View();

   9: }

View

   1: @{

   2:     ViewBag.Title = "test";

   3: }

   4:  

   5: <h2>test</h2>

   6: @ViewBag.Str

顯示

但是這不是我們要的結果。

正確的話應該是要這樣的：

@(new HtmlString(ViewBag.Str))

或著

@Html.Raw(ViewBag.Str)

   1: @{

   2:     ViewBag.Title = "test";

   3: }

   4:  

   5: <h2>test</h2>

   6:  

   7: @(new HtmlString(ViewBag.Str))
