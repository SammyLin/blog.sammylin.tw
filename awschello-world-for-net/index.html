[AWS][C#]Hello World for .NET

| Comments

我們在AWS裡面，一般來我我們會利用AWS Management Console來處理所有AWS的要處理的任務，不過當然也是可以自已寫程式來預處理任務，AWS也提供了不同的開發者，做了不同的SDK，而我是使用.NET的，當初還想說只能摸摸鼻子用其他的程式語言來做，但是沒想到AWS也為我設想(自以為~~)

以下是AWS提供的SDKs

  • SDK for Android
  • SDK for Java
  • SDK for PHP
  • SDK for iOS
  • SDK for .NET

而在前面說到，我會的是「.NET」，所以這一篇就用.NET來Hello World來一下囉，改天也來Hello World Android 跟PHP一下XD，不過Android還是Bata版，但還蠻新潮的(用手機來開電腦實體，應該很酷)

 

Hello World

1.下載AWS SDK for .NET，安裝都很容易不多做介紹，有支援VS2008跟VS2010(都有試過了，VS2005還沒試而以)

2.安裝完SDK for .NET就可以在新增專案中找到了(不過很可惜的只支援C#)，我們選擇「AWS Console Project」

3.什麼！！要輸入什麼Key呀…..不要緊張，先到Account>Security Credentials

4.把Access Key ID跟Seret Access Key照對應的地方填入，如果沒有的話，你可以按一下「Create a new Access Key」取得Access Key

5.專案建立完成，已經幫你把Access Key 寫在App.config裡面了

6.因為Amazon已經把code寫一個sample了，所以直接按「F5」進行偵錯吧。(會花一點點時間)

 
P.s 騙人~~沒有Hello World…..(誰叫Amazon 已經寫好了咩..)
